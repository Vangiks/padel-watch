# Голосовое управление счётом отложено

Рассматривали голосовое управление приложением и счётом (вплоть до полного hands-free: старт матча, начисление очков, пауза — голосом). Решили **не делать** в обозримой версии. Причины ниже специфичны для watchOS и неочевидны — поэтому фиксируем, чтобы не расследовать заново.

## Почему отложили

- **На watchOS нет встроенного распознавателя речи.** `SFSpeechRecognizer` и новый `SpeechAnalyzer` (iOS 26) на watchOS недоступны. Значит «приложение само слушает микрофон и мгновенно регистрирует *очко*» — собственного непрерывного распознавателя / хотворда / фонового микрофона построить публичными API **нельзя**.
- **Единственный hands-free путь — Siri / App Intents.** Каждая команда требует «Привет, Siri» (или raise-to-speak) + Siri-оверлей + распознавание. Фраза AppShortcut **обязана** содержать имя приложения (или его синоним), и в фразе максимум один параметр. Минимум вживую ≈ «[поднял руку] очко нам, Vamos» — несколько секунд на команду.
- **Для частого начисления очков это медленнее тапа.** Очко разыгрывается каждые ~30–60 c; одна команда Siri дольше и менее надёжна, чем тап по половине экрана. Ценность голоса именно в основном сценарии (быстрый счёт) отрицательная.
- **Конфликт с идентичностью продукта.** Standalone-runtime обещает работу на голых часах на корте, где связь часто отсутствует. Siri может требовать сеть → голос отказывал бы именно там, где приложение обещает работать. Сам инвариант голос не ломает (сетевой вызов делает система, не приложение — как с Health-синком), но завязывать ключевой ввод на best-effort-службу нельзя.

## Что взамен

Тапы остаются единственным и достаточным вводом. Для сценария «руки заняты» честнее голоса, если понадобится, зайдут **системные** механизмы без сети и без кастомного кода распознавания: Double Tap (Series 9+/Ultra 2) и кнопка действия (Ultra), привязанные к начислению очка.

## Когда пересмотреть

- Apple добавит на watchOS на-устройстве распознавание (`SFSpeechRecognizer`/`SpeechAnalyzer`) → появится быстрый офлайн-голос без Siri-латентности.
- Либо если узкий набор **редких** голосовых команд (eyes-free «какой счёт?», старт матча) окажется востребован — там латентность Siri не мешает, и это можно сделать поверх App Intents, не трогая офлайн-гарантию.
